# Imágenes Necesarias

Esta carpeta contiene las imágenes para la página de Poolly.

## Imágenes Requeridas

### 1. QR Code para descarga de app
- **Archivo**: `qr-code.png` o `qr-code.svg`
- **Tamaño recomendado**: 400x400px
- **Uso**: Sección hero, debajo del botón de descarga
- **Descripción**: Código QR que apunta al link de descarga de la app

### 2. Mockup de la app (opcional)
- **Archivo**: `app-mockup.png` o `app-screenshot.png`
- **Tamaño recomendado**: 800x1600px (proporción móvil)
- **Uso**: Se puede agregar en la sección hero o features
- **Descripción**: Captura de pantalla o mockup de la aplicación móvil

### 3. Iconos de descarga (opcional)
- **Archivos**: `app-store-badge.png`, `google-play-badge.png`
- **Tamaño recomendado**: 200x60px
- **Uso**: Botones de descarga en la sección CTA
- **Descripción**: Badges oficiales de App Store y Google Play

## Notas

- Todas las imágenes deben estar optimizadas para web
- Formato recomendado: PNG con transparencia o SVG para iconos
- Usar Next.js Image component para optimización automática
